# AutoAlias Desktop Editor 使用教程

版本：2026-05-27

## 1. 推荐工作流程

1. 点击“上传图片并提取骨架”。
2. 如果输入是普通照片或渲染图，可以勾选“原图预处理”；如果已经是线稿、ControlNet 线稿或黑底白线图，不要勾选。
3. 根据图片类型选择提取模式，必要时调整“铅笔弱线阈值”和“并线强度”。
4. 在画布上手动点选分段点，点会自动吸附到骨架上。
5. 检查蓝色预览线，确认路径正确后点击“保存当前”或“保存并下一条”。
6. 保存完所有曲线后，设置导出 Degree / 精度优先，再点击“导出 IGES / WIRE”。
7. 在 Autodesk Alias 中导入 reviewed_curves.igs，继续检查 CV 和曲率梳。

## 2. 开始区域

上传图片并提取骨架：选择图片后生成可点选的骨架与设计笔画线。

打开工程：打开之前保存的 topology_corrections.json 工程文件。

保存工程：保存当前图片、曲线、分段点和骨架修补记录。

恢复默认设置：恢复提取、显示、吸附、导出等参数，不会删除已保存曲线。

打开使用教程：打开本 PDF。

## 3. 图片与骨架提取

原图预处理：用于未处理过的照片或渲染图。系统会先估计背景和主体区域，再提取结构边缘。不适合已经是线稿的图片。

粗笔画轮廓：用于 Logo、粗马克笔、粗黑实体图形。它提取外轮廓和内孔轮廓，而不是提取笔画中心线。

提取模式：

- 自动识别：普通情况下优先使用。
- 铅笔弱线增强：适合淡铅笔、轻重不均的线稿。
- 黑底白线草图：适合黑底白线、ControlNet Canny 黑底输出。
- 白底黑线线稿：适合干净线稿、扫描图、工程线稿。
- 照片/渲染边缘：适合真实照片或渲染图，但会受反光、阴影影响。

铅笔弱线阈值：数值越低越容易保留淡线，数值越高越能抑制噪点。淡线可试 20-28，干净线稿可试 32-45。

并线强度：用于把光影造成的平行多条边缘合并成一条设计笔画。关闭最保守，强并线最激进。

按当前选项重新提取：保留当前界面，使用新参数重新生成骨架。若失败，当前已加载骨架不会被清空。

## 4. 视图与吸附

分段点吸附半径：鼠标点选或拖动分段点时，自动寻找附近骨架点的距离范围。半径过小可能吸不上，过大可能吸到旁边线。

原图：显示或隐藏底图。

完整骨架红点：显示全部骨架点，适合检查提取是否完整。

切段骨架绿线：显示经过切断后的骨架段。

设计笔画绿线：显示系统认为更接近设计笔画的连续路线。

已保存蓝线：显示已经保存的曲线。

右上角原图参考：在画布右上角显示上传原图缩略图，方便对照，不参与拟合计算。

右上角参考图透明度：调节缩略图透明度。

## 5. 手动分段

骨架修补：启用后，鼠标点击不再添加分段点，而是编辑骨架。

add：添加骨架点。

delete：删除附近骨架点。

骨架修补半径：控制 add/delete 的作用范围。

断开连续加点：停止自动连接上一处骨架修补点，下一次添加从新的位置开始。

保存当前：保存当前分段点形成的曲线。

保存并下一条：保存当前曲线，并清空当前分段点，继续画下一条。

撤回点：撤回最后一个分段点。

删除点：删除选中的分段点。

闭合开关：用于闭合曲线，例如 Logo 外圈、车轮、灯具闭合轮廓。

清空当前：清空当前正在编辑但还未保存的分段点。

## 6. 自动分段与路径候选

几何自动分段模式：

- 主线模式：更保守，优先提取长主线。
- 连续覆盖模式：会跨小断口追踪更长线。
- 局部细节模式：会提取更多短线和细节。
- 全量骨架模式：尽量把所有可追踪骨架都变成候选曲线。

几何自动分段：根据当前骨架自动生成曲线候选。工业项目中建议把它当初稿，再手动检查和修正。

分支 / 多路径候选：当两个分段点之间存在多条可选路径时，可切换候选路径。

上一候选 / 下一候选：切换当前段的路径候选。

## 7. 导出 Alias

导出 Degree：

- auto：推荐。系统自动选择 degree。
- 3：简单线、近直线。
- 5：大多数汽车设计曲线的优先选择。
- 7：复杂 S 曲线或大过渡曲线。

精度优先：用于 Logo、复杂装饰线、需要极高贴合度的情况。开启后会更重视贴合原路径，CV 美学和 Class-A 控制会弱一些。

导出 IGES / WIRE：生成 reviewed_curves.igs，并在有 Alias 转换器授权时尝试生成 reviewed_curves.wire。

打开最近导出文件夹：打开最近一次导出的目录。

## 8. 曲线列表

曲线列表显示所有已经保存的曲线。选择曲线后可以查看或删除。

全选曲线：选中列表中的所有曲线。

批量删除选中：删除选中的曲线。

## 9. 常见问题

原图预处理没有找到可用线条像素：说明当前图片不适合“原图预处理”。取消该选项，改用黑底白线草图、白底黑线线稿或铅笔弱线增强。

线条太碎：优先尝试提高并线强度，或使用设计笔画绿线辅助判断。

线条缺失：降低铅笔弱线阈值，或切换到更适合当前图片的提取模式。

点吸附到错误线：减小分段点吸附半径。

点吸不上骨架：增大分段点吸附半径。

导入 Alias 后曲线不理想：优先检查手动分段点位置。尽量在交点、设计转折点、倒角边界处点分段点。

## 10. 图片类型建议

普通照片 / 渲染图：原图预处理 + 自动识别或照片/渲染边缘。

白底黑线稿：取消原图预处理，选择白底黑线线稿。

黑底白线稿：取消原图预处理，选择黑底白线草图。

淡铅笔线：取消原图预处理，选择铅笔弱线增强，阈值 20-28。

粗 Logo / 粗笔画：勾选粗笔画轮廓。

ControlNet Canny 输出：通常不要勾选原图预处理，按黑底白线或白底黑线选择模式。
