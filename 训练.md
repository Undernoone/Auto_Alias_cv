第 3 章 面向电力通信风险预警的多参量时空联合模型设计
3.1 研究背景与总体思路

电力通信系统运行状态受到光缆沿线外部扰动、通信链路质量、设备运行工况和机房环境变化等多类因素共同影响，其监测数据通常表现出明显的多参量、长序列和强时空耦合特征。以 Φ-OTDR 分布式光纤振动信号为例，不同风险事件在空间通道、时间演化和频域响应上具有不同表现：敲击、施工破坏等突发事件往往表现为短时局部强响应；行走、挖掘、晃动等事件通常呈现连续性、节律性或准周期性振动特征；复杂户外环境下还可能出现多源扰动叠加、背景噪声增强和同类事件样本差异较大的情况。因此，单一卷积网络、循环神经网络或 Transformer 结构难以同时兼顾局部空间扰动、长程时序动态和全局语义依赖。

针对上述问题，本章在第 2 章异构数据预处理与 Φ-OTDR 时空频特征构建的基础上，设计面向电力通信风险预警的多参量时空联合模型。该模型以 Spatiotemporal-Mamba 中“多尺度空间卷积 + Bi-Mamba2 时序建模 + 门控融合”的轻量化双分支思想为核心，同时吸收 STFMambaFormer 中“时空频 Token 表示 + Bi-Mamba2 动态建模 + Transformer 全局语义建模 + 互注意力融合”的结构设计，使模型既能够提取局部空间扰动和跨通道相关性，又能够捕获长序列时间依赖和非相邻时间片段之间的全局语义关系。Spatiotemporal-Mamba 论文中采用增强多尺度 ResNet 进行空间特征提取、Bi-Mamba2 进行高效时序建模，并通过门控融合整合时空特征；STFMambaFormer 则进一步将增强后的 Token 序列输入 Bi-Mamba2 和 Transformer 两个互补分支，并通过互注意力融合获得最终表示。
